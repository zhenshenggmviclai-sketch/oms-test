# Internal OMS Test Page

This repository hosts a simple internal OMS (Order Management System) login page
for Shopee Open Platform application review purposes only.

## Purpose

- This system is an internal tool used exclusively by our own Shopee seller account.
- It is NOT a commercial SaaS product.
- It is NOT provided to third-party sellers.
- No data is collected, stored, or processed through this page.

## Login Page Notice

- The login page is a static demo interface for review verification only.
- Account and password fields are non-functional.
- No authentication logic or backend services are implemented.

## Access Scope

- Used internally by our company for product and listing management testing.
- Supports only our own Shopee store.
- No external integrations with other e-commerce platforms.

## Compliance Statement

This page is provided solely to meet Shopee Open Platform application requirements
and complies with data protection and platform usage policies.
